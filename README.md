# E-Commerce Platform

An advanced and customizable full-stack E-commerce platform built with modern web technologies[html, css, js, bootstrap, local storage]

## Features

- User authentication (login, register)
- Product catalog with categories
- Shopping cart and checkout flow
- Order management (for users, seller and admins)
- Admin panel to manage products and orders
- Responsive design
- seller panel to manage his products


### Installation

1. Clone the repository:

```bash
git clone https://github.com/m7md3bbas/E-commerce.git
cd E-commerce

```
